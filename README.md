# NCRunners — www.ncrunners.in

Website for **NCRunners**, a community of 250+ runners in and around Nanded
City, Pune, established in 2017.

Built with [Jekyll](https://jekyllrb.com/) and deployed to GitHub Pages via
GitHub Actions (`.github/workflows/pages.yml`).

## Local development

```sh
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000. See [HOWTO.md](HOWTO.md) for full
instructions on testing, deployment and editing content.

## Structure

- `index.md` — home page
- `ncrhdc*.md` — 100-Day Challenge and finisher lists
- `NC_LSOM_*.md` — NC LSoM marathon event pages
- `noSugar.md` — No Sugar Challenge
- `History.md` — how the community started
- `_layouts/`, `assets/css/` — layout and styles
